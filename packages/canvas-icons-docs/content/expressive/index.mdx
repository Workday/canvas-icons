# Expressive Icons

Expressive icons are intended to be used as simplified illustrative elements to add visual interest
without distracting your audience from their intended task. In certain scenarios they can be used to
represent specific product areas, tasks, or services. They are not intended to be actionable or to
be used as entry points between pages.
